Elementor için özel tasarım widget'ları oluşturan kapsamlı plugin. Modern ve responsive tasarımlar ile web sitenizi güçlendirin.

== Description ==

Elementor Özel Tasarım Widget'ları, web sitenizi daha etkileyici hale getirmek için özel olarak tasarlanmış widget'lar sunar. Bu plugin ile Elementor editöründe kullanabileceğiniz 5 farklı özel widget bulunmaktadır.

= Özellikler =

* **Özel Kart Widget'ı**: Modern kart tasarımları ile içeriklerinizi sunun
* **Özel Blog Widget'ı**: Son yazılarınızı etkileyici bir şekilde gösterin
* **Özel Portfolio Widget'ı**: Projelerinizi galeri formatında sergileyin
* **Özel Testimonial Widget'ı**: Müşteri yorumlarınızı güzel bir tasarımla paylaşın
* **Özel Link Meta Widget'ı**: URL'lerden otomatik meta veri çekerek link önizlemeleri oluşturun

= Link Meta Widget Özellikleri =

* Otomatik meta veri çekme (başlık, açıklama, resim)
* CORS sorunlarını çözen proxy servisleri
* Resim optimizasyonu ve WebP desteği
* Lazy loading ile performans optimizasyonu
* Responsive tasarım
* Manuel veri girişi seçeneği

= Teknik Özellikler =

* Elementor 3.0+ uyumlu
* Responsive tasarım
* SEO dostu
* Performans optimizasyonu
* Kolay kullanım
* Türkçe dil desteği

== Installation ==

1. Plugin dosyalarını `/wp-content/plugins/elementor-ozel-tasarim/` klasörüne yükleyin
2. WordPress admin panelinde 'Eklentiler' sayfasına gidin
3. 'Elementor Özel Tasarım Widget'ları' eklentisini aktifleştirin
4. Elementor editöründe widget'ları kullanmaya başlayın

= Gereksinimler =

* WordPress 5.0 veya üzeri
* Elementor 3.0 veya üzeri
* PHP 7.4 veya üzeri

== Frequently Asked Questions ==

= Elementor yüklü değilse ne olur? =

Plugin, Elementor'un yüklü olmadığı durumda uyarı mesajı gösterir ve çalışmaz.

= Widget'ları nasıl kullanırım? =

Elementor editöründe 'Özel Tasarım' kategorisinde widget'ları bulabilir ve sürükleyip bırakarak kullanabilirsiniz.

= Link Meta Widget'ı nasıl çalışır? =

URL girip 'Meta Verileri Çek' butonuna tıklayın. Widget otomatik olarak başlık, açıklama ve resim bilgilerini çekecektir.

= CORS hatası alıyorum, ne yapmalıyım? =

Plugin, CORS sorunlarını çözmek için otomatik olarak proxy servisleri kullanır. Eğer hala sorun yaşıyorsanız, farklı bir URL deneyin.

= Resimler optimize ediliyor mu? =

Evet, plugin resimleri otomatik olarak optimize eder ve WebP formatına çevirebilir.
